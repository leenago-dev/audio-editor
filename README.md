## 🎯 소개

Audio Editor는 Streamlit 기반의 직관적인 웹 인터페이스를 제공하는 음원 편집 도구입니다.
밀리초 단위의 정밀한 편집이 가능하며, 별도의 설치 없이 브라우저에서 바로 사용할 수 있습니다.

## ✨ 주요 기능

### 1. YouTube 음원 다운로드 (NEW! v3.0)
- **최고 음질 추출**: YouTube 영상에서 320kbps MP3로 오디오 추출
- **메타데이터 표시**: 제목, 업로더, 길이 정보 자동 표시
- **미리듣기**: 다운로드 즉시 재생하여 확인 가능
- **직접 다운로드 또는 편집**: 다운로드 후 바로 저장하거나 편집 탭에서 구간 편집

### 2. 정밀한 시간 제어
- **밀리초 단위 조정**: 0.001초(1ms) 단위까지 정확한 편집
- **통일된 시간 표시**: 모든 시간이 `MM:SS.mmm` 형식으로 표시
- **분/초 분리 입력**: 직관적인 시간 입력 인터페이스

### 3. 두 가지 편집 모드

#### 📥 구간 추출하기
- 선택한 구간만 별도 파일로 저장
- 하이라이트 구간, 인트로/아웃트로 추출에 최적

#### ✂️ 구간 잘라내기
- 선택한 구간을 제거하고 앞뒤를 자동으로 합침
- 불필요한 부분 제거, 광고 삭제 등에 활용

### 4. 커스텀 오디오 플레이어
- **실시간 밀리초 표시**: 재생 중 `00:00.000` 형식으로 시간 표시
- **재생 속도 조절**: 0.5x, 0.75x, 1.0x, 1.25x, 1.5x, 2.0x
- **볼륨 조절**: 슬라이더로 음량 제어
- **정밀한 탐색**: 밀리초 단위로 정확한 위치 이동

### 5. 미리듣기 기능
- 편집 결과를 다운로드 전에 즉시 확인
- 원본, 추출, 잘라내기 결과 모두 미리듣기 지원

## 🚀 설치 방법

### 필수 요구사항
- Python 3.8 이상
- pip 또는 uv 패키지 매니저

### 설치 단계

1. **저장소 클론**
```bash
git clone <repository-url>
cd audio-editor
```

2. **의존성 설치**

uv 사용 시:
```bash
uv sync
```

pip 사용 시:
```bash
pip install streamlit pydub yt-dlp
```

3. **FFmpeg 설치** (오디오 처리 및 YouTube 다운로드에 필수)

macOS:
```bash
brew install ffmpeg
```

Ubuntu/Debian:
```bash
sudo apt-get install ffmpeg
```

Windows:
[FFmpeg 공식 사이트](https://ffmpeg.org/download.html)에서 다운로드

## 💻 사용 방법

### 1. 애플리케이션 실행

```bash
streamlit run app.py
```

또는

```bash
python -m streamlit run app.py
```

### 2. 브라우저에서 접속
자동으로 브라우저가 열리며, `http://localhost:8501`에서 접속 가능합니다.

### 3. 음원 편집 프로세스

#### 방법 1: YouTube에서 다운로드

1. **YouTube 다운로드 탭 선택**
2. **YouTube URL 입력**: 영상 URL 붙여넣기
3. **다운로드 버튼 클릭**: 최고 음질(320kbps)로 자동 추출
4. **미리듣기**: 다운로드된 음원 확인
5. **선택**:
   - 바로 다운로드: 편집 없이 MP3 파일 저장
   - 편집하기: '파일 편집' 탭으로 이동하여 구간 편집

#### 방법 2: 파일 업로드

1. **파일 편집 탭 선택**
2. **파일 업로드**: MP3 또는 WAV 파일 선택
3. **원본 재생**: 업로드된 음원 확인 (총 길이 표시)
4. **구간 선택**:
   - 시작 시간: 분과 초(밀리초 포함) 입력
   - 종료 시간: 분과 초(밀리초 포함) 입력
5. **편집 모드 선택**:
   - `📥 구간 추출하기`: 선택한 구간만 저장
   - `✂️ 구간 잘라내기`: 선택한 구간 제거
6. **미리듣기**: 편집 결과 확인
7. **다운로드**: 편집된 파일 저장

### 4. 사용 예시

**예시 1: YouTube 음악 다운로드**
- YouTube URL 입력: `https://www.youtube.com/watch?v=...`
- 다운로드 버튼 클릭
- 결과: 320kbps MP3 파일 생성 및 미리듣기

**예시 2: YouTube 음원에서 인트로 추출**
- YouTube 다운로드 탭에서 음원 다운로드
- 파일 편집 탭으로 이동
- 선택: 00:00.000 ~ 00:15.500
- 결과: 15.5초 인트로 파일 생성

**예시 3: 중간 광고 제거**
- 원본: 10분 음원
- 선택: 03:00.000 ~ 04:30.000 (광고 구간)
- 결과: 8분 30초 음원 (광고 제거됨)

## 🛠 기술 스택

### Frontend
- **Streamlit**: 웹 인터페이스 프레임워크
- **HTML/CSS/JavaScript**: 커스텀 오디오 플레이어

### Backend
- **Python 3.x**: 메인 프로그래밍 언어
- **pydub**: 오디오 처리 라이브러리
- **yt-dlp**: YouTube 다운로드 라이브러리
- **FFmpeg**: 오디오 인코딩/디코딩 및 변환

### 기타
- **Base64 Encoding**: 오디오 데이터 전송
- **BytesIO**: 메모리 기반 파일 처리

## 🔧 트러블슈팅

### FFmpeg 관련 오류
```
FileNotFoundError: [Errno 2] No such file or directory: 'ffmpeg'
```
**해결방법**: FFmpeg를 설치해주세요 (위의 설치 방법 참조)

### YouTube 다운로드 오류
```
ERROR: Unable to extract video data
```
**해결방법**:
1. YouTube URL이 올바른지 확인
2. 인터넷 연결 상태 확인
3. yt-dlp 업데이트: `pip install --upgrade yt-dlp`
4. 일부 영상은 지역 제한이나 저작권으로 다운로드 불가능

### 파일 업로드 크기 제한
Streamlit 기본 설정은 200MB입니다. 더 큰 파일이 필요한 경우:

```bash
streamlit run app.py --server.maxUploadSize=500
```

### 포트 충돌
기본 포트(8501)가 사용 중인 경우:

```bash
streamlit run app.py --server.port=8502
```

### Streamlit Cloud 배포 시 FFmpeg 오류
**해결방법**: `packages.txt` 파일이 프로젝트 루트에 있는지 확인하고, 내용이 `ffmpeg`인지 확인하세요.
